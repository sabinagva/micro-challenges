# Addition Checkpoint

Create a function that takes two numbers.

Your function should return the sum of the two numbers passed in.

## Examples:

### Example 1

```js
let result = yourFunction(2, 3);
console.log(result);
```

should log 

```
5
```

### Example 2

```js
let result = yourFunction(2, 7);
console.log(result);
```

should log


```
9
```
